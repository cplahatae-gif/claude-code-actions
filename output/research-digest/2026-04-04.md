### 1. Claude Code 소스코드 51만 줄 유출: npm 패키징 실수의 후폭풍
**Tags**: #Anthropic #ClaudeCode #Security #Breaking

### 3. Claude Code 4월 업데이트: /powerup 기능·깜빡임 없는 렌더링·Named Subagents
**Tags**: #ClaudeCode #Anthropic #ProductUpdate

Anthropic은 4월에 Claude Code에 여러 주목할 만한 기능을 추가했다. `/powerup` 명령어는 애니메이션 데모와 함께 Claude Code 기능을 대화형으로 학습시키는 인터랙티브 레슨이다. `CLAUDE_CODE_NO_FLICKER=1` 환경변수로 활성화하는 깜빡임 없는 alt-screen 렌더링, `@` 멘션 자동완성에서 named subagent 지원, PreToolUse 훅에 "defer" 권한 결정 옵션(헤드리스 세션에서 중단 후 `-p --resume`으로 재개) 등이 포함됐다. 장기 세션 안정성 및 프롬프트 캐시 미스 버그도 수정됐다.
